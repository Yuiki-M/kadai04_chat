①課題番号-プロダクト名：04チャットアプリ
②課題内容（どんな作品か）：現在(STUDIOで、ですが💦)作成を進めている摂食障害Webメディアに実際に搭載する「元当事者(ピアサポーター)が現当事者のもやもや相談にお答えする質問部屋」で、QAに対して他の人が感想をコメントする仕様をイメージして作成しました
⑤工夫した点・こだわった点：時間があまり取れず相変わらず殺風景かつこだわりのない形ではありますが、毎回なんとかロジックを理解しようと悪戦苦闘しています…(そしてTutorさんに「そんなにロジックを理解しようとしなくてもよい」とたしなめていただいています…)
⑥難しかった点・次回トライしたいこと(又は機能)：実際に自分の立ち上げるサービスやメディアでもチャット機能は使えるようにしたいと思っているのですが、事務局がパトロールして不適切コメントを削除したりすることが可能にならないといけないと思ってます。今回は階層を分けずにチャットをどんどん載せていける形式にしていますが、階層を分けて特定ＱＡに対する特定コメントを限定的に削除するような構造ができないといけないと思います
⑦質問・疑問・感想、シェアしたいこと等なんでも：やはりどうしても動画を見ているだけ＋なんとなく真似して手を動かすだけではロジックが理解しきれず、一方でチューターさんからはロジックを理解しようとしても全部は無理だよ！とご助言いただくこともあり、どの程度理解をした方が良いのか、どの程度「こういうもんだ」と思って捨ておいた方が良いのか…いつも迷います。ロジックを理解できていないと再現性がないので個人的にはすごくモヤモヤしてしまい、そこが難しいです💦
そして単純にそろそろ皆さんの素晴らしいアプリとの差がすさまじくなってきたので、どこまで頑張ってついていけるかが不安です！！！(´;ω;｀)
